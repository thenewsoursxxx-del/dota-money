# DOTA MONEY — v1

Прогноз исхода матчей Tier-1 сцены Dota 2 и поиск **value против кэфа букмекера**.
Полностью статический сайт (расчёты в браузере) → бесплатно хостится на GitHub Pages и открывается как **Telegram Mini App**.
Сборка данных — на Node.js без внешних зависимостей.

## Идея

Приложение не просто показывает «кто фаворит» (это и так видно по кэфам). Оно считает **свою** вероятность исхода (Elo по недавним Tier-1 матчам) и сравнивает её с кэфом букмекера. Если модель даёт команде больше шансов, чем закладывает букмекер → это **value-ставка** (положительное матожидание). Именно так ловятся апсеты вроде Vici над Falcons или Yandex над Spirit.

## Структура

```
src/opendota.mjs   — клиент OpenDota API (Node): пагинация про-матчей, лиги, команды
src/build.mjs      — пайплайн (Node): тянет Tier-1 матчи, считает Elo → docs/data/dataset.json
docs/              — статический сайт (то, что деплоится на GitHub Pages)
  model.mjs        — общая модель: Elo, вероятность серии (BO1/BO3/BO5), value/EV/Kelly
  index.html
  styles.css
  app.js           — фронтенд, считает прогнозы client-side
  data/dataset.json  — собранные данные (создаётся build-data)
  data/upcoming.json — будущие матчи (редактируется вручную)
server.mjs         — локальный статический сервер для превью docs/
```

## Как запустить локально

```bash
# 1. Собрать/обновить данные с OpenDota (Tier-1 лиги + Elo). ~30 сек.
npm run build-data

# 2. Локальное превью
npm start   # → http://localhost:5173
```

## Деплой (GitHub Pages)

Репозиторий → Settings → Pages → Source: **Deploy from a branch**, ветка `main`, папка `/docs`.
Через минуту сайт будет доступен по `https://<user>.github.io/<repo>/`.

## Telegram Mini App

1. В [@BotFather](https://t.me/BotFather): `/newbot` → создать бота.
2. `/newapp` → выбрать бота → указать URL твоего GitHub Pages сайта.
3. Открыть мини-апп по ссылке от BotFather.

## Как добавить будущий матч

Отредактируй `docs/data/upcoming.json`. Команды можно задавать по `team_id` (надёжнее) или по названию.
`oddsA` / `oddsB` — десятичные кэфы букмекера на победу в серии; их вписываешь вручную,
а приложение само посчитает вероятность и value.

## Что это НЕ делает (пока)

Это v1. Модель — Elo по силе команд (учитывает форму, но не героев/драфт).
Дальше по плану (v2+): пулы героев игроков, матчапы/контр-драфты (пример: узкий пул Collapse под Undying),
темп/скейлинг команд, teamfight-скор, и бэктест стратегии на исторических кэфах.
